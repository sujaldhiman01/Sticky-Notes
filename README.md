# Sticky Notes App 🗒️

A simple and interactive Sticky Notes web application built using HTML, CSS, and JavaScript. This app allows users to create, edit, and delete notes while automatically saving them in the browser using localStorage so they remain available even after refreshing the page.

## 🚀 Features
- Create new sticky notes
- Edit notes in real time
- Delete notes easily
- Data persistence using localStorage
- Clean and responsive user interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ⚙️ How It Works
Users can add notes dynamically to the interface. Each note can be edited directly, and changes are automatically saved in the browser's localStorage. When the page is refreshed, the saved notes are loaded back automatically.

## 📦 Installation
1. Clone the repository
git clone https://github.com/sujaldhiman01/Sticky-Notes.git

2. Open the project folder  
3. Run `index.html` in your browser
